# Deploying a Node Js Application on AWS EC2

### Testing the project locally
1.Clone this project
 ```bash
git clone https://github.com/verma-kunal/AWS-Session.git
```
2.Setup the following environment variables - ```(.env)``` file
```bash
DOMAIN= ""
PORT=3000
STATIC_DIR="./client"

PUBLISHABLE_KEY=""
SECRET_KEY=""
```
